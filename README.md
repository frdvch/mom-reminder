# Нагадувач для мами

Локальний Android-застосунок (Kotlin, Jetpack Compose, Room). Без сервера, без інтернету
для головної функції — нагадування працюють повністю офлайн і поводяться як будильник.

## Функції
- Нагадування трьох категорій: 💊 ліки, 📅 події, 🧹 побутове.
- Повторення: одноразово / щодня / певні дні тижня / кожні N днів.
- Голосовий ввід назви нагадування (Google Voice, стандартний Android intent).
- Повноекранний алярм зі звуком і вібрацією, кнопки "Виконано" / "Відкласти на 10 хв".
- Відновлення нагадувань після перезавантаження телефону + сповіщення про пропущені.
- Щоденний дайджест: список усіх нагадувань на сьогодні о заданому в налаштуваннях часі.

## Як відкрити і зібрати
1. Встановити Android Studio (потрібно ~5-10GB вільного місця на диску для SDK).
2. File → Open → обрати цю директорію (`mom-reminder/`). Studio сама підтягне Gradle
   і запропонує встановити Android SDK потрібної версії (compileSdk 34, minSdk 26),
   якщо його ще нема.
3. Запустити unit-тести: `./gradlew test` (або через Studio: правий клік на
   `NextOccurrenceCalculatorTest` → Run) — перевіряють логіку обчислення повторень.
4. Запустити застосунок на емуляторі або реальному Android-пристрої: Run ▶.

## Обмеження емулятора при тестуванні
Емулятор Android Studio не завжди чесно відтворює:
- doze-режим (глибокий сон при вимкненому екрані) — для реальної перевірки треба
  фізичний пристрій, залишений на годину-дві з вимкненим екраном;
- реальний "холодний" перезапуск (для перевірки `BOOT_COMPLETED` краще Cold Boot
  в AVD Manager, а не просто Restart App).

Найнадійніша перевірка перед встановленням мамі — на реальному Android-телефоні:
1. Постав тестове нагадування на 1-2 хвилини вперед, заблокуй екран, дочекайся —
   має з'явитись повноекранний алярм зі звуком.
2. Перезавантаж телефон, постав активне нагадування, перевір що воно все ще спрацьовує.
3. Перевір голосовий ввід (потрібен інтернет і стандартний Google App на пристрої).
4. Увімкни дайджест на день у налаштуваннях, зачекай до заданого часу.

## Встановлення на телефон мами
1. Зібрати APK: Build → Build Bundle(s) / APK(s) → Build APK(s) (або
   `./gradlew assembleDebug` для першого тесту, `assembleRelease` для фінальної версії).
2. Перенести `app-debug.apk` (з `app/build/outputs/apk/debug/`) на телефон мами
   (USB, Google Drive, Telegram — будь-як) і встановити (дозволити "невідомі джерела").
3. При першому запуску застосунок попросить: дозвіл на сповіщення, дозвіл на точні
   алярми, і запропонує вимкнути оптимізацію батареї — на всі потрібно погодитись
   для надійної роботи.
